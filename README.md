# Bug Tracking & Test Analysis

A Java-based command-line application for tracking software bugs, managing bug statuses, and analyzing test results across features.

### Compile
```javac BugAnalysis.java```

### Run
java BugAnalysis

### Menu Options
```
1. Add Bug
2. Add Test Result
3. View Bugs
4. View Failing Features
5. Overall Pass Rate
6. Action Log
7. Update Bug Status
8. Bug Summary
0. Exit
```
